# Kaspi-Fake 🚀

Фейк приложение Kaspi (казахстанское финтех приложение)

## 🎯 Возможности

- ✅ Вход по PIN-коду (1234)
- ✅ Главная страница с меню и карточками
- ✅ Госуслуги с документами
- ✅ Навигация между страницами
- ✅ Мобильный дизайн
- ✅ React + Vite

## 📦 Установка

```bash
npm install
```

## 🚀 Запуск

```bash
npm run dev
```

Откроется на `http://localhost:3000`
